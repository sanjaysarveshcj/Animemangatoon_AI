Task 2: Sentiment Analysis on User Comments

Overview:
This task involved performing sentiment analysis on user comments for the article "The Difference Between Manga And Manhwa" using the TextBlob library. The aim was to determine whether comments were positive or negative.

Approach:
Data Collection: Extracted user comments from the specified article.
Sentiment Analysis: Used TextBlob to analyze the sentiment of each comment and categorize them as positive or negative based on polarity scores.
Results Summary: Calculated and presented the percentage of positive vs. negative comments.

Code Documentation:
Each section of the code is well-commented to describe its functionality.
The final output includes a summary of the sentiment analysis results.

Usage:
Run the script to analyze the comments and view the summary of sentiment distributions.
